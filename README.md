# Snake Game

Implementation of the classic Snake game developed with HTML, CSS and JavaScript.

## Features

<ul>
  <li>Snake movement</li>
  <li>Food generation</li>
  <li>Score tracking</li>
  <li>Collision detection</li>
</ul>

## Technologies Used

<ul>
  <li>HTML</li>
  <li>CSS</li>
  <li>JavaScript</li>
</ul>

## Purpose

This project was developed to practice programming logic, game mechanics and JavaScript fundamentals.
